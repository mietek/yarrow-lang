Yarrow Installation Guide for binaries

Install Yarrow

This is easy:

  1. Get the correct version for your system by ftp, and place it in a directory, say /usr/yarrow/. The filename is in the format yarrow-architecture-os.ext.
    • yarrow-bin-sparc-sun-sunos4.tar.gz (approx. 900 K).
      This versions seems to work under Solaris as well. Try at your own risk...
  2. Go to that directory and unpack it.
    • For Unix-like systems:
      gzip -cd yarrow-bin-architecture-os.tar.gz | tar xf -
  3. Set the environment variable YARROW. (You should include this in your .login file)
    • For Unix-like systems: e.g.
      setenv YARROW /usr/yarrow
  4. Add the directory containing the binaries to your path, (you should include this in your .login file)
    • For Unix-like systems: e.g.
      setenv PATH ${PATH}:/usr/yarrow/bin
      Users of csh shells should type
      rehash
      to make this path extension take effect.
    Alternatively, place the binaries in a directory in your path.

Run Yarrow

Now you can start yarrow with the command
yarrow

Test Yarrow

Start it, and you should get something like 

Welcome to Yarrow v0.99
Type 'help introduction' for an introduction.
>

then type the lines behind the prompt ("> " and "$ ") 

> prove test : @P:*. P -> P
--------------------------------------------------
@P:*. P->P

$ intros
P : *
H : P
--------------------------------------------------
P

$ assumption
Goal proved!

$ exit
prove test : @P:*. P->P
intros
assumption
exit

test := .. : @P:*. P->P
> help help
Syntax: 1. help
        2. help <command>
Effect: The first variant ...

If you get different output something has gone wrong.

That's it!
